    vue 利用pdfjs-dist插件转图片

    # 需求

    公司的仓库系统electron框架有个添加个点击打印pdf的功能，用iframe (opens new window)发现行不通（浏览器内核不行），采用的是pdf.js完成这个需求 其中踩了不少坑 记录一下

    # 实现

    # 安装依赖

    官方网址：
    可以引入cdn 尝试一下
    也可以下载到本地 引入也可以
    网络上大部分的人都比较支持安装@2.2.228这个版本
    但是目前来说 最新版没有什么特别大的bug 直接下最新的也可（反正到最后都要改东西的😊）

    npm install --save pdfjs-dist

    # 引入
    <script>
const PDFJS = require('pdfjs-dist');
//我要修改的是这
let workerSrc = require('pdfjs-dist/build/pdf.worker.entry');
PDFJS.workerSrc = workerSrc;
//有一些是要修改这
PDFJS.GlobalWorkerOptions.workerSrc = require('pdfjs-dist/build/pdf.worker.min');
</script>

    # 执行代码
    //画步准备 我直接用dom操作 创建个画布出来 vue不建议哈
function _renderPage (num) {
  pdfDoc.getPage(num).then((page) => {
    let canvas = document.createElement('canvas');
    var vp = page.getViewport({scale: 1});
    let ctx = canvas.getContext('2d')
    let dpr = window.devicePixelRatio || 1
    let bsr = ctx.webkitBackingStorePixelRatio ||
      ctx.mozBackingStorePixelRatio ||
      ctx.msBackingStorePixelRatio ||
      ctx.oBackingStorePixelRatio ||
      ctx.backingStorePixelRatio || 1
    let ratio = dpr / bsr
    let viewport = page.getViewport({scale: window.innerWidth / vp.width});
    canvas.width = viewport.width * ratio
    canvas.height = viewport.height * ratio
    canvas.style.width = viewport.width + 'px'
    ctx.setTransform(ratio, 0, 0, ratio, 0, 0)
    let renderContext = {
      canvasContext: ctx,
      viewport: viewport
    }
    page.render(renderContext);
    document.body.appendChild(canvas);
  })
}
function _loadFile (url) {
  PDFJS.getDocument(url).promise.then((pdf) => {
    pdfDoc = pdf
    pages = pdfDoc.numPages
    _renderPage(1);
  });
}

    # 注意点

    1.pdf预览有大小/缩放/不清晰的问题？
    ✋我目前的解决方式是page.getViewport({scale:1.2}) 大概给个范围 可能每个都不一样 但是page.getViewport这块可以调试一下
    2.爆什么worker的警告⚠？
    ✋我没理...反正我跑得动就可以了
    3.能传什么形式的pdf进去处理?
    ✋我传过本地/在线网址 看了源代码是可以传二进制的 可以试试 base64我试不行
    4.报错 Warning: Error during font loading: The CMap “baseUrl“ parameter must be specifie
    参考链接
    ✋到node_modules->pdfjs-dist->build->pdf.js源代码中
    ctrl+F 搜索cMapUrl 后面添加两行

    params.rangeChunkSize = params.rangeChunkSize || DEFAULT_RANGE_CHUNK_SIZE;
  params.CMapReaderFactory = params.CMapReaderFactory || _display_utils.DOMCMapReaderFactory;
  params.ignoreErrors = params.stopAtErrors !== true;
  params.fontExtraProperties = params.fontExtraProperties === true;
  params.pdfBug = params.pdfBug === true;

//添加的两句
params.cMapPacked =true 
params.cMapUrl ='https://cdn.jsdelivr.net/npm/pdfjs-dist@2.2.228/cmaps/'
