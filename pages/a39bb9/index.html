vuepress-theme-vdoing做个人博客

# 介绍

基于vuepress做的一个主题
vuepress-theme-vdoing官网 https://doc.xugaoyi.com/ (opens new window)
vuepress官网 https://www.vuepress.cn/ (opens new window)
‼️ 注意
建议先看一遍vuepress目录然后再去vuepress-theme-vdoing中进行初步操作

# 前期准备

gitHub账号
node版本大于14(我用的最新版，若版本过低的话需要卸载掉node再到官网安装)
yarn/npm

# 安装

1.选择自己想要发风格进行安装 我选的是知识库兼博客

# 拷贝项目

git clone https://github.com/xugaoyi/vuepress-theme-vdoing.git (opens new window) //如果不能下载成功的话 使用 git clone git@github.com:xugaoyi/vuepress-theme-vdoing.git

# cmd 运行

cd vuepress-theme-vdoing

# install dependency 注意：如安装不成功请关闭淘宝源。

npm install # or yarn install

# develop

npm run dev # or yarn dev

# 部署到gitHub中

# 更新到gitHub中

  1. 在本地登陆自己的git邮箱账号
  2. 右键文件夹 点击git bush 在弹窗中依此输入

git add . git commit -m '备注' git push

# 问题

# 1.没有自动生成目录

想要点击关于后会弹到目录页面 侧边框


解决：

在00.目录页中配置所需要的生成的目录

.vuepress>config.ts 记得写好文件名前序号，不然不会生成
